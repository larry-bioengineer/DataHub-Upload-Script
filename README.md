# LoRA local server

## What is this
A flask server for collecting local sensory data and send to cloud. 

## data storage
configuration of the devices (e.g. their IP addresses) will be stored in a file device_config.json
### Data example for the devices config

| Type         | Gateway_IP     | Gateway_Port | Device_ID |
|--------------|---------------|--------------|-----------|
| temperature  | 192.168.1.130 | 502          | 1         |
